### Project Setup
1. Clone this repository:
   ```bash
   git clone https://github.com/Manideep-Codes/Razorpay_Visual_Studio_Code_Extension.git
   cd razorpay-vscode-extension
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Compile Extension
   ```bash
   npm run compile
   ```
