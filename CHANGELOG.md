# Changelog

All notable changes to the "Razorpay for VS Code" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-12-17

### Added

- **AI-Powered Assistant**: Get intelligent help with Razorpay integration using OpenAI or Google Gemini
- **Code Snippets**: Ready-to-use code snippets for common Razorpay operations
  - Create orders
  - Verify payments
  - Process refunds
  - Handle webhooks
- **SDK Integration**: One-click SDK installation with automatic project detection
  - Support for Node.js, Ruby, Python, PHP, Java, Go, and .NET
- **Trigger Events**: Test webhook events directly from VS Code
- **MCP Server Integration**: Advanced integration capabilities
- **Multi-language Support**: Snippets available for TypeScript, JavaScript, Ruby, Python, and more

### Features

- Razorpay Activity Bar with dedicated views
- Configurable AI providers (OpenAI and Google Gemini)
- Secure credential storage in VS Code settings
- Project type auto-detection
- Hover documentation for Razorpay API methods
